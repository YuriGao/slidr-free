# Slidr Free

Slidr Free is an open-source macOS utility for edge gestures, middle click, fine control, smart typing detection, and cursor freeze workflows.

This project is independent and non-affiliated with any similarly named commercial products or vendors.

## Requirements

- macOS 13 or later
- Swift 5.9 or later

## Build and test

```bash
swift build
swift test
```

## Unsigned build warning

Development builds are unsigned. macOS may require you to explicitly allow the app in Privacy & Security settings before running it.

## Features

- Volume edge gesture
- Brightness edge gesture
- Middle click
- Fine control
- Side swapping option
- Bottom-quarter-only option
- Smart typing detection
- Cursor freeze
